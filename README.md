# Portfolio BTS SIO

Ce dépôt contient un mini-site PHP présentant un portfolio pour un étudiant en BTS SIO (option SLAM). Le site comprend :

- `index.php` : page d'accueil présentant le profil et les objectifs.
- `veille.php` : veille technologique orientée informatique quantique et cybersécurité.
- `stages.php` : récapitulatif des deux stages réalisés.
- `code_conduite.php` : code de conduite professionnel.
- `partials/` : entête et pied de page partagés.
- `assets/css/style.css` : feuille de style principale.

## Prérequis

- PHP 8.0 ou supérieur.

## Lancer le site en local

```bash
php -S localhost:8000
```

Ensuite, ouvrez [http://localhost:8000](http://localhost:8000) dans votre navigateur.

## Personnalisation

- Remplacez `[Votre Nom]` dans `index.php` par votre nom.
- Mettez à jour les coordonnées dans `partials/footer.php`.
- Ajoutez vos ressources de veille ou projets personnels selon vos besoins.
